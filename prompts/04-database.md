# Prompt 04: Database Design

## Prompt Used

> Design the complete database schema for LinkPeek with 9 tables: Users, ApiKeys, Previews, Collections, Tags, PreviewTags, CollectionPreviews, ApiUsage, Screenshots.
>
> Generate:
> 1. ER Diagram with all relationships
> 2. Complete PostgreSQL DDL with indexes, triggers, RLS policies
> 3. Prisma schema with proper type mappings
> 4. Relationship documentation with cascade strategies
> 5. Indexing strategy justified by query patterns
> 6. Full-text search using PostgreSQL tsvector
> 7. Migration plan with Prisma Migrate workflow

## Result

Complete database design with 9 tables, 28 indexes, 7 triggers, 11 RLS policies, full-text search, and seed data. See `docs/phase-3/`.
