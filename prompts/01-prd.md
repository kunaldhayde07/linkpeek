# Prompt 01: Product Requirements Document

## Prompt Used

> You are a Principal Software Architect and Product Manager. Generate a comprehensive Product Requirements Document (PRD) for a "Link Preview API Platform" called LinkPeek.
>
> Include:
> - Executive Summary
> - Problem Statement
> - Product Vision and North Star Metric
> - Target Audience (4 segments)
> - Core Features (MVP) with priority levels (P0/P1/P2)
> - Out of Scope items
> - Success Criteria
> - Constraints (free-tier infrastructure only)
> - Risks & Mitigations
> - Release Plan (Alpha → Beta → GA)

## Result

Generated a complete PRD with 19 features, 4 target audience segments, risk matrix, and phased release plan. See `docs/phase-1/01-PRD.md`.
